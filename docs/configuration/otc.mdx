## OTC Configuration

### Cost Allocation

OpenCost will automatically detect OTC as the cloud service provider (CSP) by detecting the node label `cce.cloud.com/cce-nodepool`. When OTC is detected as the CSP, OpenCost attempts to retrieve pricing data from the [OTC Pricing API](https://calculator.otc-service.com/en/open-telekom-price-api-ui/).

No API key is required to retrieve the public pricing data.
